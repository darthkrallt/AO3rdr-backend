AO3rdr Bookmarker

A bookmark manager for reading on Archive of Our Own (archiveofourown.org). Available on both Firefox desktop and mobile beta.

Get it now!
    Features include:
  • Three star rating system
  • Hide works
  • Blacklist by tags
  • Bookmark by chapter read
  • Scan bookmarks for updates
  • Keep track of last visit
  • Backup and restore user data
  • Cloud Sync for multiple browsers
  • Open source code!
